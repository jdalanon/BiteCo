# ☕ Bite Co.

A simple, responsive coffee shop website built using **HTML**, **CSS**, and **JavaScript**.
This project showcases a modern landing page for a small coffee business with a simple yet attractive layout and responsive design.

## 🌟 Features

* Responsive navigation bar
* Hero section with call-to-action
* Coffee menu cards with Buy Now button
* Cart section 
* About Us section
* Contact Us form
* Smooth scrolling navigation
* Simple animations using JavaScript
* Mobile-friendly layout

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6/ES2015+)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/jdalanon/BiteCo.git
```

2. Open the project folder.

3. Launch `index.html` in your web browser.

No additional installation or dependencies are required.
